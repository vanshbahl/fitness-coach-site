# Project Overview: Quick Strength

## Project Vision
Quick Strength is a premium fitness coaching application built for Abhay Pandey. The application discards the traditional "landing page + form" model in favor of an interactive, app-like onboarding experience. The booking process *is* the product. Users are guided through a sleek, multi-step assessment that feels like joining an exclusive program, culminating in seamless payment, automated calendar integration, and immediate access to the Quick Strength community.

## Business Goals
- **Maximize Conversion**: Replace intimidating long-forms with engaging, bite-sized onboarding steps.
- **Premium Brand Perception**: Every interaction, slider, and micro-animation should feel like an elite fitness app (think Nike, Linear, Gymshark).
- **Zero Friction**: Achieve sub-90 second booking times optimized entirely for one-handed mobile use (Instagram traffic).
- **Total Automation**: Automatically handle scheduling, payments, and follow-up communications.

## User Journey
1. **Instagram Arrival**: The user is already warm and interested in Abhay's coaching.
2. **Landing Hero**: Immediately greets the user with an immersive "Start Trial" CTA.
3. **Interactive Assessment**: A beautiful, multi-step wizard capturing personal details, body metrics, goals, fitness level, and equipment.
4. **Review & Payment**: A polished summary followed by secure Razorpay checkout (₹49).
5. **Confirmation & Onboarding**: A celebratory success screen, calendar invite integration, and a direct link to join the WhatsApp community.

## MVP Scope
The MVP focuses purely on the onboarding funnel, seamless conversion, and administrative management.
- **Onboarding Wizard**: 12-step interactive flow with progress tracking, visual cards, and sliders.
- **Booking Backend**: Robust API to capture assessment data, verify Razorpay payments, and sync with Google Calendar.
- **Coach Dashboard**: An admin panel for Abhay to review applications, mark attendance, track payments, and manage active trial enrollments.

## Future Scope
- Client portal for workout and nutrition tracking.
- Recurring subscription logic for enrolled students.
- Automated WhatsApp notifications throughout the lifecycle.
