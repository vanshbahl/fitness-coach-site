# Frontend Tech Stack

- **Framework**: React 19
- **Bundler**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **State Management**:
  - Global/Server State: TanStack Query (React Query)
  - Form State: React Hook Form + Zod
- **Routing**: React Router v7
- **Deployment**: Vercel
